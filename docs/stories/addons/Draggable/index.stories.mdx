# Draggable

<Source language="jsx" code="import { Draggable } from 'bycoders-ui-react'" />

The `Draggable` component is a wrapper for `dragging/dropping` things. It should be used
always wrapped by `React-DnD` wrapper to work as expected.

<br></br>

## Default

In most of cases you should use it in this way:

<Preview
  mdxSource={`
const DraggableContainer = () => {
  const items = [
    { id: 1, type: 'card', content: 'foo', position: 1 },
    { id: 2, type: 'card', content: 'baz', position: 2 },
  ];
  const [stateItems, setStateItems] = useState(items);
  const handleOnMove = (position: OnMoveType) => {
    console.log(position)
  };
  return (
    <DndProvider backend={HTML5Backend}>
      {stateItems.map((item, idx) => (
        <Draggable
          key={item.id}
          position={idx}
          type={item.type}
          onMove={handleOnMove}
        >
          <div>
            {item.content}
          </div>
        </Draggable>
      ))}
    </DndProvider>
  );
};
export default DraggableContainer;
`}
