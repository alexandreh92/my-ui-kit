## Default

    name="Default"
    args={{

      weight: 'header-500',

## Render As

You can add a property named `renderAs` to change the component wrapper:

<Preview>
  <Story
    name="Custom Wrapper"
    args={{
      children: 'Some String',
      weight: 'header-500',
      renderAs: 'h1',
    }}
  >
    {Template.bind({})}
  </Story>
</Preview>
